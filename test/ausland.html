🇺🇸 USA

Spezielle Arten von Anmeldungen

Eine provisional application ist eine einfache, kostengünstige Voranmeldung, die ein Prioritätsdatum sichert, aber nicht geprüft wird und nie selbst zu einem Patent führt. Spätestens nach 12 Monaten muss eine nicht-provisorische (non-provisional) Anmeldung eingereicht werden, um die Priorität zu erhalten.

Eine divisional application wird eingereicht, wenn das USPTO eine Uneinheitlichkeit (Restriction Requirement) feststellt oder der Anmelder selbst die Anmeldung aufgrund mehrerer unabhängiger Erfindungen aufteilen will. Die Teilanmeldung erhält denselben Zeitrang wie die ursprüngliche Anmeldung und ist vergleichbar mit einer EP-Teilanmeldung.

Eine continuation application enthält keinen neuen Gegenstand gegenüber der ursprünglichen nicht-provisorischen Anmeldung. Sie erlaubt jedoch, die Prüfung „fortzuführen“, z.B. um weitere Anspruchssätze zu verfolgen, wenn die ursprüngliche Anmeldung kurz vor der Erteilung steht.

Eine continuation-in-part application (CIP) enthält neuen zusätzlichen Inhalt gegenüber der Elternanmeldung. Der alte Inhalt behält das ursprüngliche Prioritätsdatum, der neue Inhalt erhält das neue Anmeldedatum der CIP.
